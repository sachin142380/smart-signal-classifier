# 🚀 AI Signal Analyzer

🔥 A powerful **AI-powered signal classification web app** built using Machine Learning, capable of analyzing and classifying multiple waveform types in real-time.

---

## 🌐 Live Demo

👉 https://smart-signal-classifier-xeupycs2znm74sqqmsjrtk.streamlit.app

---

## ✨ Features

* 📊 Classifies **15+ signal types** (sine, square, chirp, noise, etc.)
* ⚡ Real-time signal generation & prediction
* 📈 Frequency analysis using FFT
* 🎧 Audio upload & classification
* 📊 Confusion matrix visualization
* 🎨 Clean interactive UI with tabs

---

## 🧠 Tech Stack

* 🐍 Python
* 🌐 Streamlit
* 🤖 Scikit-learn
* 📊 NumPy & SciPy
* 📈 Matplotlib & Seaborn

---

## 🚀 Run Locally

```bash
git clone https://github.com/sachin142380/smart-signal-classifier.git
cd smart-signal-classifier
pip install -r requirements.txt
streamlit run app.py
```
